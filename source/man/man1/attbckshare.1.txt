NAME
attbckshare - attach network share for backups
SYNOPSIS
attbckshare [OPTION]
DESCRIPTION
Attaches a pre-determined network CIFS or NFS share as a target location for backup
files. Two target locations are supported; a weekly target for the full weekly,
differential and incremental type backups, and, the ad hoc type for the full
ad hoc and selective type backups.
OPTIONS
-a, --adhoc	attaches the pre-defined adhoc backup target share.

-h, --help  prints usage information

-v, --verbose  produce verbose output

-V, --version  prints version information

-w, --weekly	attaches the pre-defined weekly target backup share
EXIT STATUS
0
if OK,

1
if minor problems,

64
error in command line arguments.

65
failed to mount share due to mount error.

66
Backup share already mounted.
