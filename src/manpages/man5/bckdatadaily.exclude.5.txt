NAME
bckdatadaily.exclude - Daily data backup exclusions file.
DESCRIPTION
bckdatadaily.exclude contains one line for each file or directory to be skipped during a bckdatadaily backup.
FILES
bckdatadaily.exclude	The file is stored in an etc directory. The exact location depends on the method of installation but for Autotools installations on Linux this is likely to be /usr/local/etc/backups/bckdatadaily.exclude.
