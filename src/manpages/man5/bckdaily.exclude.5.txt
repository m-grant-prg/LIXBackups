NAME
bckdaily.exclude - Daily incremental / differential data backup exclusions file.
DESCRIPTION
bckdaily.exclude contains one line for each file or directory to be skipped during a daily backup.
FILES
bckdaily.exclude	The file is stored in an etc directory. The exact location depends on the method of installation but for Autotools installations on Linux this is likely to be /usr/local/etc/backups/bckdaily.exclude.
